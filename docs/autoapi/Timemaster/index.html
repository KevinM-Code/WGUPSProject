class Timemaster.Timemaster¶
calc_truck_time(self, distance)¶

This method takes a parameter (the distance that was traveled by the truck) and with the given miles per hour value accumulates how much time, down to the second, the truck needs to make the trip, converts it and returns a datetime.timedelta object.

Note

Time-complexity is O(1).

Parameters

distance (float) – The distance the truck drove on its route

Returns

The time the truck took to finish its route

Type

datetime.timedelta

time_converted_to_timedelta(self, time)¶

This is a converter method that takes an argument, converts and returns the argument value as hours, minutes and seconds datetime.timedelta() object.

Note

Time-complexity is O(1).

Parameters

time (string) – in the format 00:00:00 to signify hrs:mins:sec

Return type

datetime.timedelta

military_to_normal_time(self, time_string)¶

This method changes military time format to normal am/pm time format

Example:

>>> print (Timemaster.military_to_normal_time('13:00:00'))
01:00:00 PM

Note

Time-complexity is O(1).

Parameters

time_string (string) – military time in the format 13:01:57

Return type

string
