DataImporter.location_names¶

An instance of class HashMap

DataImporter.distance_data¶

The file WGUPSDistances.csv information is read into the variable, then the csv data read in, is organized as a 2D list().

Note

The time-complexity of list() is O(N)

DataImporter.first_7_letters_to_ascii_value(text)¶

This converts the first 7 characters, including spaces if they exist, to a large integer comprised of all the ascii codes concatenated producing a unique dictionary key.

Note

The time-complexity of this method is O(1)

Parameters

text (string) – The string you would like to convert

Returns

ascii numbers concatenated

DataImporter.locations¶

The file WGUPSPlacesOfDelivery.csv information is read into the variable, then the csv data read in is organized into DataImporter.location_names.

Note

Utilizing a HashMap of the locations was my attempt to minimize time complexity for lookup. The time complexity is O(N)
