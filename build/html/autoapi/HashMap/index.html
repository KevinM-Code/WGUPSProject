class HashMap.HashMap¶

This HashMap has an insertion, deletion, and search time-complexity is O(N). However on average, time-complexity is a good Θ(1).

_calc_hash(self, key)¶

This private method takes and int and returns what bag number it will be put into.

Note

Time-complexity is O(1).

Parameters

key (int) – The key value

Returns

bag number

Return type

int

set(self, key, value)¶

This method sets the key-value pair in the HashMap.

Note

This method has a time-complexity of O(N)

Parameters

  • key (int) – ID for the value

  • value (list or any) – A list of information

get(self, key)¶

With the key given as a parameter, this method returns the value assigned to the key.

Note

This method has a time-complexity of O(N)

Parameters

key (int) – ID for the value

Returns

A list of information

Return type

list or any

remove(self, key)¶

This method removes the key value pair from the HashMap based on the key.
