total_mileage(self, row, col, total_dist)¶

This method refers to the list that was created from the file WGUPSDistances.csv. Only the lower left triangle is filled in so the method tries to find a value by row and column, if no value is found it swaps the row and column values to find the distance between two locations. The method then adds the value looked up, to the total_dist value coming in as a parameter.

Note

This method has a time-complexity of O(1)

Parameters

  • row (int) – The row number

  • col (int) – The column number

  • total_dist (float) – This gives the method a looping accumulating capability (the distance is returned and can come right back in)

Returns

the mileage between the queried locations plus whatever has already been added.

Return type

float

get_best_route(self, location, packages_info_onboard)¶

Here is the workhorse of the application. It takes the starting point location and an array of packages on board packages_info_onboard. For every location, it finds the shortest route to the next location and continues till the end of the route, attempting to reorder the packages to a shorter route. The method then returns the packages and locations in the most efficient order a greedy algorithm can produce.

Note

This method has a time-complexity of O(N^2)

Parameters

  • location (int) – starting location

  • packages_info_onboard (list) – an array of packages on board

Returns
